## ğŸ†˜ TROUBLESHOOTING

**Images not showing?**
- Check file names match EXACTLY (case-sensitive!)
- Make sure images are in same folder as HTML

**Sounds not playing?**
- Check audio folder exists
- Check file names in audio folder match exactly
- Try clicking somewhere on page first (browsers require user interaction)

**Multiplayer not syncing?**
- Everyone must be on the same network OR use hosted version
- If using file:// URLs locally, multiplayer won't work between computers
- Solution: Use GitHub Pages or Netlify

**Game not loading?**
- Try a different browser (Chrome works best)
- Clear browser cache
- Check browser console for errors (F12)
